This file explains all the steps needed to run this web application named Concerto. For details have a look at the project report.

Required software: 
	- Node.js: https://nodejs.org/en/download
	- Express: https://expressjs.com/
	- Yarn: https://classic.yarnpkg.com/lang/en/docs/install/#mac-stable

To start the backend go to /backend and run following commands:
$ yarn
$ yarn dev

To start the frontend go to /frontend and run the following commands:
$ yarn
$ yarn dev

Ideally you should start the backend before the frontend, otherwise there is no data to use.
